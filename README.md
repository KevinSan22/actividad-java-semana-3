# actividad-java-semana-3
# Actividad Java sin IDE

## Descripción
Programa en Java que solicita datos de un vehículo y los muestra en pantalla.

## Compilación
javac Vehiculo.java

## Ejecución
java Vehiculo

## Requerimientos funcionales
- Ingreso de datos
- Visualización de datos

## Requerimientos no funcionales
- Uso de Java
- Ejecución en consola
- Sin IDE
